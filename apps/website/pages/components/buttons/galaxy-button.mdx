# Props

| Prop        | Type          | Required | Description                                               |
|-------------|---------------|----------|-----------------------------------------------------------|
| `folderName`| `string`      | `Yes`      | The name of the folder to be displayed on the button.    |
| `images`    | `string[]`    | `Yes`      | An array of image URLs to be displayed when the button is hovered over. |
